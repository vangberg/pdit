# CodeMirror 6 Documentation

This directory contains comprehensive documentation for CodeMirror 6, downloaded for offline reference and agent use.

## Contents

### Main Documentation
- **guide.md** - Complete system guide covering architecture, data model, view system, and extension development
- **api-reference.md** - Overview of all core packages and their purposes
- **extensions.md** - Catalog of all available core extensions
- **migration-v5-to-v6.md** - Migration guide from CodeMirror 5 to 6
