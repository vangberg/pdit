# pdit Features & Capabilities

## Core

- **Output-focused** - Results appear inline next to the code that generated them
- **Just Python scripts** - No notebooks, no cells, no special format. Work with plain `.py` files
- **File watching** - Changes to the file on disk automatically reload in the editor
- **Auto-run on save** - Execute code automatically when the file changes
- **Works with coding agents** - Perfect companion for Claude Code, Cursor, and other AI coding tools that edit files

## Rich Output

- **IPython kernel** - Persistent namespace, Jupyter-like workflow
- **Matplotlib plots** - Inline image rendering
- **Interactive DataFrames** - Sortable, searchable tables
- **Markdown rendering** - Format text output with markdown
