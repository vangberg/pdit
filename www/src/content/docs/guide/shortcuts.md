## Execution

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + Enter` | Execute the entire script |

## Editor

pdit uses CodeMirror, which supports standard text editing shortcuts:

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + Z` | Undo |
| `Cmd/Ctrl + Shift + Z` | Redo |
| `Cmd/Ctrl + A` | Select all |
| `Cmd/Ctrl + D` | Select next occurrence |
| `Cmd/Ctrl + /` | Toggle comment |
| `Tab` | Indent |
| `Shift + Tab` | Outdent |

## Navigation

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + G` | Go to line |
| `Cmd/Ctrl + F` | Find |
| `Cmd/Ctrl + H` | Find and replace |
