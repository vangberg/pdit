## Features

<CardGrid stagger>
  <Card title="Inline Results" icon="document">
    See execution results displayed inline, grouped by the lines of code that produced them.
  </Card>
  <Card title="DataFrames" icon="list-format">
    Pandas DataFrames render as clean, scrollable tables directly in your editor.
  </Card>
  <Card title="Plots & Images" icon="star">
    Matplotlib plots and images appear inline without any extra configuration.
  </Card>
  <Card title="IPython Powered" icon="rocket">
    Full IPython kernel support with magic commands and rich output types.
  </Card>
  <Card title="File Watching" icon="magnifier">
    Edit files in your favorite editor and see updates in real-time.
  </Card>
  <Card title="Export" icon="open-book">
    Export your scripts with results to standalone HTML files for sharing.
  </Card>
</CardGrid>

## Quick Install

```bash
pip install pdit
```

Then run `pdit script.py` to get started.
