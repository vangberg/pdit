## UI style

UI STYLE: Terminal/Newslist

- Monospace fonts (font-mono)
- Grayscale palette with minimal accent color
- Dense spacing: py-0.5, px-1, text-xs
- No rounded corners, simple borders (border-gray-300)
- Text-based icons: ▶▼·□⚡
- Uppercase labels: TREE, CONTENT:, OUTPUT:
- Plain buttons: border border-gray-400 bg-white
- Pre-formatted text for output
- Minimal hover states (hover:bg-gray-100)
- Information-dense, compact layouts
- Show metadata (IDs, types, counts)
